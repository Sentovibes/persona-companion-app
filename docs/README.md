# Persona Companion — Web

A browser-based companion for Persona 3, 4, and 5. No install required.

**Live site:** https://sentovibes.github.io/persona-companion-app/

**Android app:** https://github.com/Sentovibes/persona-companion-app/releases/latest

---

## What's included

- **Personas** — browse, search, and filter all personas across P3FES, P3P, P3R, P4, P4G, P5, and P5R
- **Fusion Calculator** — find every recipe for any persona, verified against megaten-fusion-tool data
- **Enemies / Shadows** — stats, resistances, drops, and boss phases
- **Social Links / Confidants** — full rank guides with dialogue choices and point values
- **Classroom Answers** — every exam and quiz answer for P3, P4, and P5

## How to use

Just open the site in any browser. Pick a series, pick a game, and go.

Works on mobile too — the layout adapts to your screen size.

## Android app

The web version covers the core features. The Android app adds:
- HD images (1500+ persona and enemy portraits, optional 1.3GB download)
- Offline support
- Favorites and recently viewed
- Tablet / TV layouts

Download the latest APK from the [Releases page](https://github.com/Sentovibes/persona-companion-app/releases/latest).
