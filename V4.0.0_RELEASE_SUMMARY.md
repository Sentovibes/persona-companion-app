# Version 4.0.0 Release Summary

## 🎨 Major Feature: Image Support (BETA)

Version 4.0.0 introduces profile images for personas and enemies!

### What's Included

✅ **542 unique images** integrated
- 246 persona images
- 296 enemy images
- No duplicates

✅ **Profile picture display**
- Circular images next to names
- Tap to expand to full screen
- Adaptive sizing (phone/tablet/TV)

✅ **Works offline**
- All images stored in app assets
- No internet required
- ~15MB additional app size

### ⚠️ BETA Limitations

**Many images are missing:**
- Personas: 87.5% coverage (171 missing)
- Enemies: 52.7% coverage (911 missing)
- Overall: 67.2% coverage (1082 missing)

**Quality issues:**
- Some images may be incorrect
- Some from older SMT games (not Persona)
- Automated scraping may have errors
- No manual verification done yet

## Build Instructions

```bash
# Clean build
gradlew.bat clean

# Build debug APK
gradlew.bat assembleDebug

# Build release APK
gradlew.bat assembleRelease
```

APK location:
- Debug: `app/build/outputs/apk/debug/app-debug-4.0.0.apk`
- Release: `app/build/outputs/apk/release/app-release-4.0.0.apk`
