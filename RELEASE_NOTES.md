# Release Notes - v5.0.0

## 🎉 Major Release - HD Images & Multi-Device Support

### ✨ New Features

**HD Image System**
- Optional 1.3GB HD image pack with 1500+ persona and enemy images
- Download directly from app or import local ZIP file
- Images automatically hidden when not downloaded (no placeholders)
- Profile pictures in phone mode, full-size images in tablet/TV mode

**Multi-Device Optimization**
- Phone mode with compact layouts and touch navigation
- Tablet mode with dual-pane layouts
- Android TV mode with D-pad navigation and large text
- Chromecast support for streaming to TV

**Enhanced Filtering**
- DLC content toggle (show/hide DLC personas)
- Episode Aigis toggle (P3R/P3FES The Answer content)
- P3P protagonist selection (Male MC or FeMC social links)

**Social Features**
- Share personas and enemies with full stats
- Share social links with dialogue choices
- Share boss phases and multi-part bosses

### 🐛 Bug Fixes
- Fixed image loading in list screens
- Fixed P5/P5R classroom answers (removed incorrect exclusive flags)
- Fixed social link data for all games
- Improved ZIP extraction with proper path handling
- Fixed state persistence for downloaded images

### 🎨 UI Improvements
- Adaptive layouts for all screen sizes
- Improved text scaling for accessibility
- Better error handling and user feedback
- Cleaner settings screen with organized sections

### 🔧 Technical Improvements
- Migrated to cloud-based image delivery
- Optimized APK size (13MB without images)
- Improved performance and memory usage
- Better error logging and diagnostics
